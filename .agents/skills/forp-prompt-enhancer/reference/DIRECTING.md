# FILM DIRECTING & VISUAL STORYTELLING — operator reference

Dense, teachable synthesis (StudioBinder, No Film School, Filmmakers Academy, Columbia Film Glossary,
Wikipedia, craft blogs; 2024–2026). Director's checklist + how each rule maps to AI start/end-frame
clip authoring at the end. Film grammar is evergreen; the load-bearing part for us is §C.

## 1. The director's job & visual storytelling
- **Show, don't tell.** Emotion/plot/relationship via image, action, staging — not exposition. Subtext
  (what's beneath) carries more than the literal.
- **Motivate every shot & move** — each answers "why this, why now?": reveal/conceal, walk us through
  space, track action, or produce a feeling. Unmotivated moves pull us out. Best moves are barely
  perceptible but felt (slow push-in = rising interiority).
- **Framing = status.** Angle/size/screen-position encode power before a word.
- **POV / eyeline** (look → what they see) binds us to a character.
- **Visual motifs** — a recurring image/object/color/shape that reinforces theme and can EVOLVE in
  meaning as the character changes. (Ours: glass — shatters → carries the fragments → reforged in the sun.)

## 2. Blocking & staging
- *Blocking* = actor paths; *Staging* = actors + objects + camera placement. **Blocking drives the
  camera** (design movement so the camera must go somewhere — kills arbitrary coverage).
- Pass: (1) emotional geography, (2) sketch positions/power, (3) layer FG/MG/BG depth, (4) let blocking
  dictate framing, (5) only motivated movement.
- Techniques: **depth staging** (read multiple performances, spatial hierarchy), **triangle blocking**
  (3 actors = dynamics/allegiance that shifts as the triangle shifts), **status via position** (seated
  vs standing; who holds the symbolic prop), **mirrors/reflections** keep off-screen presence.

## 3. Continuity & spatial grammar (invisible editing)
- **180° rule / axis of action:** draw the line through the two subjects; keep the camera on ONE side
  → each keeps a consistent screen side (A looks right, B looks left). Travel keeps one screen
  direction. **Cross the line only deliberately** (disorientation/power-flip) and re-establish via:
  (a) move the camera across on-screen, (b) a neutral on-axis/head-on shot, (c) a cutaway, (d) a
  character re-blocks the line.
- **30° rule:** successive shots of the same subject change ≥30° in angle and/or size, else = jump cut.
  Break only to *create* a jump (emphasis/unease).
- **Eyeline match** — look and reverse directionally consistent (she looks screen-right → target is
  screen-right of her).
- **Screen direction** — same side per character; **exit right → enter left** to read as continuous
  travel; left→right feels natural, right→left feels like struggle.
- **Match on action** — cut on a continuing movement so motion flows unbroken (the universal invisible cut).
- **Eye trace** — keep the point of interest in the same screen region across a cut.
- **Establishing shot** — fix location/space before going tight.

## 4. Coverage
- **Master** (wide, whole action, the spatial anchor) + **reverses/OTS** (opposite sides of the axis)
  + **singles** + **inserts** (CU of the key object) + **cutaways/reactions**. Shoot the master first
  to lock geography, then tighter — all on one side of the line.
- **Minimum for a beat:** establish space + a closer angle for emotion + any insert the beat needs to be legible.

## 5. Shot design & composition
- **Sizes & emotion:** EWS (isolation/scale) · WS (context) · FS (full-body action) · MWS/cowboy
  (action+emotion) · MS (neutral dialogue buffer) · MCU (facial nuance) · CU (intensity/turn) · ECU
  (visceral / a critical detail). Wider = objective/distant; tighter = subjective/intense — escalate to
  tighter as emotion peaks.
- **Angles:** eye-level (neutral/peer) · low (power/threat/heroism) · high (vulnerability) · dutch
  (disorientation/unease).
- **Lens:** wide = exaggerated depth, amplified z-move, face distortion up close, immersion; long/tele
  = compression (stacked planes), flatten/isolate via shallow focus, "running but going nowhere" =
  futility. Keep lens character consistent within a scene.
- **Composition:** rule of thirds (off-center = isolation; centered = confrontation/control) · symmetry
  = order/entrapment · headroom (too much sinks, too little claustrophobic) · **lead/nose room** in the
  direction faced/moved (remove it = entrapment) · negative space = loneliness/dread · leading lines ·
  frame-within-a-frame = confinement/surveillance.

## 6. Pacing & rhythm within a scene
- **Shot duration is the primary tempo control:** short+frequent = urgency/chaos; long/held =
  weight/intimacy/dread. Movement inside the frame also raises energy.
- **Build & release:** accelerate cuts + tighten sizes to a peak, then release (wider shot / held beat /
  a "breath" of stillness). **Contrast = emphasis** (a long CU after rapid cutting lands hardest).
- **Cut on motivation / match on action** (hide the cut); cut on a static frame only to make it noticed.

## 7. Action / fight directing
- **Establish geography first** — a wide that says who is where; return to it so we never lose placement.
- **Hold the axis / hero line** — hero one side, opponent the other, so blows read directionally;
  re-establish the new line when choreography rotates them.
- **Cause → effect legibility** — show the strike AND the impact/recoil; MS keeps interaction intimate
  without losing placement.
- **Impact frames & rhythm** — cut fast on power reversals (but each shot purposeful & readable; don't
  dissolve geography). **Camera height encodes power** (low up at the winner, high down on the pinned);
  flip as power shifts.
- **Structure like a scene:** wind-up (slow) → escalation → climax; never open at full throttle.

## C. HOW EACH RULE MAPS TO AI START/END-FRAME CLIPS (our core)
Each clip = one *shot*; a scene = a chain of clips; AI has NO innate memory of the axis, so each frame
must enforce it.
- **Hold the 180° axis across the chain.** Decide the axis once; in every clip's start AND end frame
  keep each character on their assigned screen side. To change it, author a bridging clip whose camera
  crosses on-screen, or a neutral head-on clip, then reset for all later clips.
- **Screen direction & eyeline across seams.** Traveling subject keeps one direction; exit-right in
  clip N's end frame → enter-left in clip N+1's start frame. Dialogue: A's clips look right, B's look left.
- **Match-on-action seam vs intentional cut.** For a *continuous* action across two clips, make clip
  N's END frame ≈ clip N+1's START frame (pose/position/framing) → invisible join. For a *cut* to a new
  angle, make the next start frame differ ≥30° → reads as an edit, not a jump-cut glitch.
- **Coverage as a clip set.** A beat = one wide/establishing clip (locks geography) + tighter clips
  (reverses/singles/inserts) all on the same axis/direction/eyelines; generate an ECU insert to make a
  beat unmistakable.
- **Composition per frame.** Escalate size to CU/ECU as emotion peaks; low angle = dominant, high =
  vulnerable; consistent headroom + correct lead room; thirds (off-center = isolation, centered =
  confrontation); wide-lens for immersion, long-lens compression for isolation/dread — consistent within a scene.
- **Pacing via clip length & motion**; **action chains**: establish wide → hold the hero line →
  strike-clip then impact/reaction clip → return to wide on rotations → flip height on power reversals.
- **Eye trace across seams** — keep the key subject/detail in the same screen region from one clip's
  end frame to the next's start frame.

## DIRECTOR'S CHECKLIST (violate only for a deliberate, felt effect)
1. Stay one side of the 180° line; re-establish when crossing. 2. Change ≥30° between same-subject
shots. 3. Consistent screen direction (exit right → enter left). 4. Match eyelines. 5. Cut on
motivation / match on action. 6. Motivate every shot & move; wider=distance, tighter=intimacy,
escalate with emotion. 7. Establish before tight. 8. Consistent headroom/lead room. 9. Keep action
geography legible (cause+effect both on screen). 10. Preserve eye trace. **Meta-rule:** break a rule
only when the emotional intent IS the break; an accidental break is a mistake, a felt one is a choice.

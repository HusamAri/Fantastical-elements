# SCREENWRITING & STORY — scenario craft for short cinematic pieces

Operator reference for writing the STORY before any shot. A shot is only as good as the beat it
serves. For FORP we mostly write **wordless** cinematic pieces (title sequences, hero intros), so
story is carried by image, motion, motif and sound — not dialogue.

## 1. Structure (know the skeleton, then feel it)
- **Three acts:** Setup (~25%) → Confrontation/escalation (~50%) → Resolution (~25%). Two turning
  points hinge the acts.
- **Eight-sequence method:** each act splits into 2–3 sequences, each its own mini-movie with a
  question and a button. Useful to keep momentum in a 60–120s piece: think 4–6 short sequences.
- **Save the Cat beats** (feature %, scale to any length): Opening Image → Theme Stated → Setup →
  Catalyst → Debate → Break into Two → B-Story → Fun & Games (the "promise of the premise") →
  Midpoint (false peak/valley) → Bad Guys Close In → All Is Lost → Dark Night → Break into Three →
  Finale → **Final Image** (rhymes with the opening).
- **Hero's Journey (compressed):** ordinary world → call → threshold → trials → ordeal → reward →
  return with the elixir. For a hero intro, "trials → ordeal → reward" is the spine.

## 2. Short-form & title-sequence storytelling
- **Write the ENDING first** (the 3-second final image). Everything aims at it. Ours: sun-blinder →
  settle → CTA "create your own fragment."
- **Hook in the first seconds** (≤15s): the most characteristic image + a question. Ours: the calm
  cast + "Six were witnessed."
- **Escalation arc that always works:** **calm → threat → transformation → aftermath.** Map beats to
  it; each beat must raise stakes or reveal.
- **One idea per beat.** A title sequence is a string of clear images, not a plot. Motif > exposition.
- **Removal test:** if a beat can be removed and nothing emotional/logical breaks, cut it.

## 3. Character & emotion (visual, no dialogue)
- **Want vs Need:** the external goal vs the internal truth. Hero: *want* = keep the six fragments
  whole; *need* = the fire he carries is strength, not damage.
- **Convey character through ACTION + choice under pressure**, not description. What he protects,
  what he refuses, how he takes a hit and answers.
- **Stakes** must be legible: what is lost if he fails (the fragments shatter / the people are lost).
- **Theme** as a visual motif that recurs and pays off (glass that shatters → glass that carries the
  fragments → the whole reforged in the ultimate).

## 4. Scene construction
- **A scene turns on a value shift** (+ → − or − → +). If nothing changes, it's not a scene.
- **Enter late, leave early** — start at the last possible moment, cut on the turn.
- **Conflict** in every beat (an obstacle, a cost, an opposing force — here the hooded attacker, the
  damage, the interrupt).
- **Button** each beat with an image that launches the next.

## 5. Logline & premise
- Logline = [protagonist] + [want] + [obstacle] + [stakes] in one sentence. Test it; if the premise
  isn't compelling in a sentence, the film won't be.
- **"So what?" test** + **removal test** (does the piece justify existing; does each beat earn its place).

## 6. BEAT-SHEET TEMPLATE — <120s wordless cinematic hero piece
| # | Beat | Function | Escalation | On-screen/motif | Sound |
|---|------|----------|-----------|-----------------|-------|
| 1 | Opening image / hook | establish hero + world + question | calm | signature object/power | motif in, low bed |
| 2 | Inciting image | the world changes; stakes appear | threat | the break/loss | impact |
| 3–N | Rising beats (sequences) | each raises stakes / reveals a facet | build | one motif each | layered SFX |
| mid | False peak or lowest point | turn | reversal | power or wound | shift |
| — | Ordeal / all-is-lost | biggest pressure | crisis | the hit / the last drop | drop-out / silence |
| — | Transformation | hero answers; need realized | surge | the super-state | swell |
| — | Climax | the definitive act | peak | the ultimate | full |
| — | **Final image (write first)** | resonance; rhymes with opening | aftermath | settle | tail |
| — | CTA / turn to audience | invitation / button | — | brand line | out |

## FORP hero intro mapped (N=17)
Opening (calm cast) → inciting (shatter) → rising beats woven with the six fragments + fight moves
(Twin Spiral, Federica, dodge) → ordeal (interrupt = discs destroyed) → last drop (Başak) →
transformation (Flame Inside) → quiet-before (Baver, Yaşar) → climax (Ancient Sunlight) → final image
(settle) → CTA. Escalation calm→threat→transformation→aftermath is intact.

## STORY-TEST CHECKLIST (before shotlisting)
- [ ] Ending written first; opening rhymes with it.
- [ ] Hook lands in ≤15s with zero prior knowledge.
- [ ] Clear want + need + stakes.
- [ ] Every beat raises stakes or reveals; passes the removal test.
- [ ] Escalation calm→threat→transformation→aftermath present.
- [ ] Theme carried by a recurring motif that pays off.
- [ ] One idea per beat; wordless-legible.
